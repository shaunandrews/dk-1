### File Locations

|------|-------|

| Shared Packages | `wordpress/plugins/ciab-admin/packages/` |

### Common Commands

```bash
# Install dependencies
pnpm install
composer install

# Build all plugins
pnpm build:all

# Development mode
pnpm dev

# Start WordPress environment
pnpm env:start

# Run tests
pnpm test:e2e
pnpm test:unit
```
